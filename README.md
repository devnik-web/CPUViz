# CPUViz
A web-based simulator built with HTML, CSS, and JavaScript that lets visualize multiple CPU scheduling algorithms. It supports FCFS, SJF (Preemptive &amp; Non-Preemptive), Priority (Preemptive &amp; Non-Preemptive), Round Robin.It generates separate Gantt charts and calculates Completion Time, Turnaround Time, Waiting Time, and averages for each algorithm.
